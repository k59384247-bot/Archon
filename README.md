# Agentic BIM Modeller

An agentic system that generates BIM (Building Information Modeling) and architectural floor plans, combining a fine-tuned SAM-based data pipeline, constraint-aware planning agents, and parametric geometry/export engines. The system takes design intent as input and produces validated, code-compliant floor plans exportable to industry-standard formats (IFC/DXF).

This project is being built phase-by-phase according to the specification in [docs/](docs/), starting with Phase 0 (environment and repository setup).
